## Design e Stile
- **Palette**: Toni terrosi e naturali — verde salvia, beige caldo, bianco crema, accenti oro/ambra
- **Font**: Sans-serif moderno e leggibile (Inter o simile) + serif per titoli eleganti
- **Layout**: Sezioni full-width, molto padding, scroll fluido
- **Immagini**: Loto, natura, silhouette yoga — toni caldi e rilassanti
- **Mobile First**: deve funzionare perfettamente su smartphone
- **Animazioni**: Subtle fade-in on scroll, niente di aggressivo

## Regole obbligatorie
1. Il sito è interamente in **inglese**
2. NON aggiungere framework CSS (no Bootstrap, no Tailwind) — CSS puro
3. Tutto in un singolo `index.html` con CSS e JS separati
4. Le immagini vanno nella cartella `img/`
5. Responsive design obbligatorio
6. Performance: il sito deve caricarsi in meno di 2 secondi
7. Accessibilità base: alt text, contrasto colori, navigazione da tastiera
